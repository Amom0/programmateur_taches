Lancer le programme en tapant "make" dans la console :
	- Rentrez le path de la commande que vous voulez utiliser dans le terminal
	- Rentrez ensuite la commande en elle-même
	- Rentrez le délai en minute
	- Puis rentrez le nombre d'itération voulu

Cas d'utilisations :

1er cas :
	/bin/ls
	ls -l
	1
	3 

2ème cas :
	/bin/cp
	cp README.txt ..
	2
	1
